# Web Scraper

The ALTPRESS Web Scraper allows users to view and save their favorite current news articles from the ALTPRESS website. The user is also able to clear all articles and scrape new ones. <br>

 
 ### Technologies Used

* HTML
* CSS
* Bootstrap
* JavaScript
* MongoDB
* MongoJS
* Mongoose
* Express
* Handlebars
* Cheerio
* Axios
* Node
* Heroku
